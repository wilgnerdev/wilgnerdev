## 📌 Projeto em destaque

🏛️ **Portal de Cursos — Fundação O Pão dos Pobres (HTML + CSS)**  
Portal web institucional desenvolvido como **Trabalho de Conclusão de Curso (TCC)** do programa de aprendizagem profissional na Fundação O Pão dos Pobres.

O objetivo é oferecer uma interface acessível e clara para que alunos e interessados possam visualizar os cursos disponíveis na instituição. O projeto está publicado via GitHub Pages e conta com:

* Estrutura semântica em HTML5;
* Estilização responsiva com CSS3;
* Navbar responsiva desenvolvida separadamente como componente de estudo;
* Aplicação real entregue para uma organização social em Porto Alegre.
